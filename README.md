## Usage

1. Select a scenario
2. Choose sample size (5-20)
3. Optionally enter a seed for reproducible data
4. Click "Genereer Dataset"
5. Follow the steps to calculate correlation

## Development

This is a pure HTML/CSS/JavaScript application with no dependencies.

To run locally, simply open `index.html` in any modern browser.
